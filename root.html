Étapes pour rooter un Pixel 7 Pro

- PC : Se rendre sur developers.google.com/android/ota - Pixel : Aller dans les paramètres, "à propos de l'appareil", et trouver le numéro de build PC : Télécharger l'image OTA correspondant exactement au numéro de build - Pixel : Activer le mode développeur - Pixel : Activer le débuggage USB - Pixel : Activer le débuggage OEM - Brancher le Pixel au PC - PC : Choisir de le connecter à Linux - PC : Lancer le terminal en copiant collant cette url dans un nouvel onglet: Effectuer la commande pour voir si le Pixel est identifié : adb devices - PC : Mettre le téléphone en fastboot avec la commande : adb -d reboot bootloader - PC : Remettre la connexion à Linux et vérifier la connexion avec la commande : fastboot devices - PC : Débloquer le bootloader avec la commande : fastboot flashing unlock - PC : Relancer le téléphone avec la commande : fastboot reboot - Pixel : Reconfigurer pour activer le mode développeur, USB et OEM - PC : Extraire deux fois l'image OTA du .zip - PC : Aller dans le dossier "images" et copier et transférer "init_boot.img" sur le Pixel - PC : Transférer Magisk sur le Pixel - Pixel : Installer Magisk - Pixel : Ouvrir Magisk - Pixel : Cliquer sur installer en haut - Pixel : Sélectionner init_boot.img - Pixel : Cliquer sur "continuer" Le fichier patché sera dans le dossier téléchargement du téléphone et devrait être nommé "magisk_patched[...].img" - Pixel : Ouvrir Magisk - PC : Transférer le fichier du Pixel au PC dans le dossier platform-tools sur Linux - Pixel : Redémarrer le téléphone, en maintenant Volume bas et Power pour le passer en mode Fastboot - PC : Effectuer la commande pour voir si le Pixel est toujours identifié : fastboot devices - PC Effectuer la commande : fastboot flash init_boot magisk Et appuyer sur tab pour automatiquement trouver le fichier Cela devrait envoyer, et écrire sur 'init_boot_a' ou 'init_boot_b' et devrait marquer 'Finished' Pixel : Sélectionner et appuyer sur Start Le téléphone affichera un avertissement du bootloader, et devrait s'allumer normalement Pixel : Ouvrir Magisk et vérifier si Magisk est bien installé
